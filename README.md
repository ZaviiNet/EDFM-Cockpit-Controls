# EDFM-Cockpit-Controls

A lightweight, browser-based tool for visualizing and customizing Elite Dangerous keybindings.

Live Demo 

## üöÄ Features

* **Visual Matrix: See your keybinds mapped onto a HUD-style keyboard layout.**

* **HOTAS Support: Automatically detects and lists bindings for X52, Joysticks, and Gamepads.**

* **Drag & Drop: Upload your own Custom.4.2.binds file to see your specific layout.**

* **Edit & Export: Remap keys directly in the browser and export a .binds file ready for the game.**

* **Local Storage: Automatically saves your changes so you can come back later.**

* **Frontier API Authentication: Login with your Frontier account to save your keybinds to our servers.**

## üõ†Ô∏è How to use

**Open the website. [here](https://cockpitcontrols.edfm.space)**

The "Default" profile loads automatically.

To view your own controls, click Load .binds and select your file (usually found in %LOCALAPPDATA%\Frontier Developments\Elite Dangerous\Options\Bindings).

To change a bind, toggle Edit Mode, click a row, and press the new key.

Click Export to download your updated file.

Installation / Hosting

This is a static web application consisting of a single HTML file and a JSON data file.

Clone this repository.

Open index.html in any modern web browser.

No server or installation required.
